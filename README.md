# SQL JOINs in BigQuery

## Project Overview

This project demonstrates how to use SQL JOIN operations in Google BigQuery using employee and department datasets. The goal of this project is to understand how relational databases combine data from multiple tables using common columns.

The project includes examples of:

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* FULL OUTER JOIN

# SQL JOIN Queries Included

## INNER JOIN

Returns only matching records from both tables.

## LEFT JOIN

Returns all records from the left table and matching records from the right table.

## RIGHT JOIN

Returns all records from the right table and matching records from the left table.

## FULL OUTER JOIN

Returns all matching and non-matching rows from both tables.
